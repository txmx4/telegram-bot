### Install (termux)
Clone this project

```bash
> pkg install git
> git clone https://github.com/txmx4/telegram-bot_2
> cd telegram-bot_2
```

Install the dependencies:

```bash
> pkg install python
> python install.py
```

### Usage
run the telegram bot

```bash
> python start.py
```

### Install (windows)
install nodejs in [here](nodejs.org/)

```bash
> npm install
> npm install axios
> npm install telegraf
> npm install nodemon
> nodemon bot.js
> atau
> node bot.js
```
